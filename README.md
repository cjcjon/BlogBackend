Koa를 기본으로 한 Rest Server Backend 코드입니다. 현재 http://118.67.132.126/ 에서 운영중입니다.

## 사용한 라이브러리
- [koa](https://koajs.com/)
- [koa-body](https://github.com/dlau/koa-body)
- [koa-router](https://github.com/koajs/router)
- [mariadb](https://github.com/mariadb-corporation/mariadb-connector-nodejs)
- [aws-sdk](https://github.com/aws/aws-sdk-js)
- [dotenv](https://github.com/motdotla/dotenv)
- [joi](https://github.com/sideway/joi)
- [mime-types](https://github.com/jshttp/mime-types)
- [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)
- [bcrypt](https://github.com/kelektiv/node.bcrypt.js)
- [ip-address](https://github.com/beaugunderson/ip-address)
- [request-ip](https://github.com/pbojinov/request-ip)
- [sanitize-html](https://github.com/apostrophecms/sanitize-html)
- [uuid](https://github.com/uuidjs/uuid)
- [dayjs](https://github.com/iamkun/dayjs)
